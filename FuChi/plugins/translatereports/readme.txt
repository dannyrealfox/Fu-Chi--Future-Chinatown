tested up to: 2.0
author: Brian Herbert
author website: http://www.brianherbert.com

== Description ==
The "Create Report" page will have a new field for a translation of a report.

== Installation ==
1. Copy the entire /translatereports/ directory into your /plugins/ directory.
2. Activate the plugin.
