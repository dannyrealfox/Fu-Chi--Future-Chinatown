== Description ==
This plugin provides notification in the admin panel when the database version in the code doesn't match the version in the database. This is useful for administrators that frequently update their deployments and may inadvertently forget to run the latest database update script.

== Installation ==
1. Copy the entire /versionnotifier/ directory into your /plugins/ directory.
2. Activate the plugin.
